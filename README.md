# Student Management Portal

A full-stack Student Management Portal built using PHP, MySQL, HTML, and CSS.  
This project allows admins to manage student records, marks, and attendance efficiently.

## Features

- Admin Login / Logout
- Add Student Details
- View Students List
- Edit Student Information
- Delete Student Records
- Search Students by Name / Email
- Add Student Marks
- View Marks Report
- Mark Attendance
- Clean Dashboard UI

## Tech Stack

- PHP
- MySQL
- HTML
- CSS
- XAMPP

## Setup Instructions

1. Install XAMPP
2. Start Apache and MySQL
3. Import `student_portal.sql` in phpMyAdmin
4. Move project folder to `htdocs`
5. Open browser:
   http://localhost/student_portal/
